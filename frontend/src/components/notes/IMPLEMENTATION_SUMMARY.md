## 技術仕様

### スタイリング

- **フレームワーク**: Tailwind CSS
- **選択状態**: `border-pink-500 ring-2 ring-pink-500`
- **ホバー状態**: `hover:border-pink-300 hover:scale-105`
- **グリッド**: `grid-cols-4 md:grid-cols-6 lg:grid-cols-8`

### パフォーマンス最適化

1. **React.memo**: コンポーネント全体をメモ化
2. **useMemo**: フィルタリング結果をメモ化
3. **lazy loading**: 画像の遅延読み込み
4. **width/height**: レイアウトシフト防止

### アクセシビリティ

- `aria-label`: 各ボタンに設定
- `alt`: 画像に適切な代替テキスト
- キーボード操作: ボタン要素で対応

### レスポンシブデザイン

- **モバイル（<768px）**: 4列グリッド
- **タブレット（768px-1024px）**: 6列グリッド
- **デスクトップ（>1024px）**: 8列グリッド

### 設計判断

1. **React.memo使用**: 親コンポーネントの再レンダリング時の最適化
2. **useMemo使用**: 検索クエリ変更時のみフィルタリング実行

- 初回レンダリング: 全チャンピオン表示でも高速
- 検索時: useMemoによる最適化で即座に反応
- スクロール時: lazy loadingで必要な画像のみ読み込み
