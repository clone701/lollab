# NoteForm コンポーネント実装サマリー

## 概要

NoteFormは、チャンピオンノートの作成・編集に使用する再利用可能なフォームコンポーネントです。

## 実装完了項目

### ✅ 7.1 フォームの基本構造を実装

- **ノートタイプ選択UI**: ラジオボタンで「汎用ノート」「対策ノート」を選択
- **マイチャンピオン選択UI**: ChampionSelectorコンポーネントを使用
- **敵チャンピオン選択UI**: ChampionSelectorコンポーネントを使用（条件付き表示）
- **戦略メモ入力欄**: textareaで10,000文字まで入力可能
- **保存・キャンセルボタン**: フォーム送信とキャンセル機能

### ✅ 7.2 ノートタイプによる表示切り替えを実装

- **対策ノート**: 敵チャンピオン選択UIを表示
- **汎用ノート**: 敵チャンピオン選択UIを非表示
- **自動リセット**: ノートタイプを「対策」→「汎用」に変更すると、敵チャンピオンIDを自動的にnullにリセット

### ✅ 7.3 フォームバリデーションを実装

- **ノートタイプの検証**: 'general' または 'matchup' のみ許可
- **マイチャンピオンの必須検証**: 未選択時にエラーメッセージ表示
- **敵チャンピオンの条件付き必須検証**: 対策ノートの場合のみ必須
- **メモの文字数制限検証**: 10,000文字まで（textareaのmaxLength属性で制限）
- **エラーメッセージ表示**: バリデーションエラーをリスト形式で表示

### ✅ 7.4 レスポンシブデザインを実装

- **モバイル**: `flex-col` で縦積みレイアウト
- **デスクトップ**: `md:flex-row` で横並びレイアウト
- **チャンピオン選択UI**: 画面サイズに応じて自動調整

## コンポーネントAPI

### Props

```typescript
interface NoteFormProps {
  /** 初期値（編集モードの場合） */
  initialValues?: Partial<ChampionNote>;
  /** フォーム送信時のハンドラー */
  onSubmit: (data: Partial<ChampionNote>) => void;
  /** キャンセル時のハンドラー */
  onCancel: () => void;
  /** ローディング状態 */
  loading?: boolean;
  /** 編集モードかどうか（編集モードの場合、ノートタイプを変更不可にする） */
  isEditMode?: boolean;
}
```

### 使用例

#### 作成モード

```tsx
<NoteForm onSubmit={handleSubmit} onCancel={handleCancel} loading={loading} />
```

#### 編集モード

```tsx
<NoteForm
  initialValues={existingNote}
  onSubmit={handleSubmit}
  onCancel={handleCancel}
  loading={loading}
  isEditMode={true}
/>
```

## 主要機能

### 1. ノートタイプ切り替え

- ラジオボタンで「汎用ノート」「対策ノート」を選択
- 編集モードではノートタイプを変更不可（disabled）
- ノートタイプ変更時に敵チャンピオンIDを自動リセット

### 2. チャンピオン選択

- ChampionSelectorコンポーネントを使用
- マイチャンピオンは常に表示
- 敵チャンピオンは対策ノートの場合のみ表示
- レスポンシブレイアウト（モバイル: 縦積み、デスクトップ: 横並び）

### 3. バリデーション

- `validateNote` 関数を使用してクライアント側バリデーション
- エラーメッセージをリスト形式で表示
- バリデーション成功時のみ `onSubmit` を呼び出し

### 4. ローディング状態

- ローディング中は全ての入力要素を無効化
- 保存ボタンのテキストを「保存中...」に変更
- キャンセルボタンも無効化

### 5. 文字数カウント

- メモ入力欄の下に「X / 10,000 文字」を表示
- リアルタイムで文字数を更新

## スタイリング

### カラースキーム

- **背景**: グレー系（bg-gray-800, bg-gray-900）
- **テキスト**: ホワイト/グレー系（text-white, text-gray-200）
- **アクセント**: ピンク系（bg-pink-600, focus:ring-pink-500）
- **エラー**: レッド系（bg-red-900, border-red-700）

### レスポンシブブレークポイント

- **モバイル**: デフォルト（768px未満）
- **タブレット/デスクトップ**: `md:` プレフィックス（768px以上）
