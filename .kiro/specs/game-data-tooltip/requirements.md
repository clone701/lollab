# メモ: game-data-tooltip（未着手）

## 概要

チャンピオン・スキル・アイテム・ルーンの画像にホバーするとツールチップを表示する共通UIコンポーネント。ノート閲覧・チャンピオン検索など複数の機能から利用される。

## やりたいこと

### 共通Tooltipコンポーネント
- 任意の画像要素にホバーするとツールチップを表示
- `components/ui/` に配置し、どこからでも使える共通コンポーネントとして実装

### ツールチップの内容
- **チャンピオン**: チャンピオン名・ロール・簡易説明
- **スキル（Q/W/E/R/パッシブ）**: スキル名・説明・コスト・クールダウン
- **アイテム**: アイテム名・説明・コスト・ステータス
- **ルーン**: ルーン名・説明

### 利用箇所（予定）
- 汎用ノート閲覧画面のコマンド埋め込み（`/yasuo/Q` など）
- チャンピオン検索機能のスキル表示
- 対策ノートのルーン・アイテム選択UI（将来）

## 前提条件（着手前に必要なもの）
- Data Dragon（dragontail）の解凍・整備が完了していること
- スキル・アイテム・ルーンの説明データをJSONで整備済みであること
- `public/images/` 配下に各画像が配置済みであること

## ノートのコマンド記法（game-data-tooltipと連携）
- `/yasuo` → チャンピオンアイコン（general-note-createで実装済み）
- `/yasuo/Q`, `/yasuo/W`, `/yasuo/E`, `/yasuo/R`, `/yasuo/passive` → スキル画像
- `/item/3157` → アイテム画像
- `/rune/8112` → ルーン画像
