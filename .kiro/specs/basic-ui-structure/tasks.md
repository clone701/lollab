# 実装計画: 基本UI構造

## 概要

この実装計画は、LoL Labアプリケーションの基本UI構造を構築します。Next.js 15のApp Routerを使用し、ナビゲーションバー、フッター、グローバルローディング、認証統合を含む全ページ共通のレイアウトを確立します。

## 注意事項

- 既存の`app/providers.tsx`、`app/api/auth/[...nextauth]/route.ts`は変更しない
- GIFアニメーションは通常の`<img>`タグを使用（Next.js `<Image>`は非対応）
- 全てのコンポーネントは日本語コメントを含める
- TypeScriptの型安全性を確保
- Tailwind CSSを使用したスタイリング
- アクセシビリティ要件（alt属性、lang属性等）を満たす
