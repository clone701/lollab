# 実装計画: ノート編集・削除機能

## 概要

既存のNoteFormコンポーネントを拡張し、ノートの閲覧・編集・削除機能を実装します。NoteCardにクリックイベントを追加し、NotesPageで一覧・閲覧・編集の3つの状態を管理します。
